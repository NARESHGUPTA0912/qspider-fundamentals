# Table in HTML:
-> it is coll. of row and columns
-> to create table in html we use <table></table> tag.
-> here we arev creating row by rows.
-> to create table row we use <tr></tr> tag.
-> to create table cells html provide two tags.
   i) <th></th> tag. it is use to create table heading cells.
   ii) <th></th> Tag:- it is used to create table heading cells.

-> to provide title or caption to the table we use <caption></caption> tag.
-> caption tag we have to write within table tag.

# Attribute of th and td tag:
1. rowspan: it is an att. used to span the cells on row basis.
2. colspan: it is an att. used to span the cells on col basis.

Examples:- 
<th rowspan="2" colspan="3"></th>
<td rowspan="2"></td>
<th colspan="4"></th>

# Attribute of table tag:
1. cellpadding
2. rules

# thead: 
-> the top most part of our table we have to write within <thead></thead> tag.
# tfoot: 
-> the bottom most part of our table we have to write within <tfoot></tfoot> tag.
# tbody: 
-> the remaining content of our table we here to write within <tbody></tbody> tag.

# Imp. of thead, tfoot & tbody tag:
it is higher recommended to used inside table for better explanation of code and crawler's understanding purpose.
